# Habit Tracker

A habit tracking application built using Streamlit, SQLite, Plotly, and APScheduler.

## Features

- Create and manage habits
- Daily habit completion tracking
- Streak calculation
- Achievement badges
- Progress visualization using Plotly
- Automated reminders

## Tech Stack

- Python
- Streamlit
- SQLite
- Plotly
- APScheduler

## Installation

1. Clone the repository

```bash
git clone <repository-url>
cd habit_tracker
```
2. Install dependencies

```bash
pip install -r requirements.txt
```
3. Run the application

```bash
streamlit run app.py
```
